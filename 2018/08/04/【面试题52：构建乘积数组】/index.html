【面试题52：构建乘积数组】

Aug 4, 2018

构建乘积数组.

题目描述

给定一个数组A[0,1,…,n-1],请构建一个数组B[0,1,…,n-1],其中B中的元素B[i]=A[0]A[1]…A[i-1]A[i+1]…A[n-1]。不能使用除法。

解题思路

//    ---------------------
// B0 | 1  | A1 | A2 | A3 |
//    ---------------------
// B1 | A0 | 1  | A2 | A3 |
//    ---------------------
// B2 | A0 | A1 | 1  | A3 |
//    ---------------------
// B3 | A0 | A1 | A2 | 1  |
//    ---------------------
//以上面的4*4数组为例
//B0 = A1*A2*A3,B1 = A0*A2*A3,B2 = A0*A1*A3,B3 = A0*A1*A2
//第一轮，先算左下角的元素，即
//B0 = 1, B1 = A0, B2 = A0*A1, B3 = A0*A1*A2
//第二轮，再端右上角的元素，即
//B3 = 1, B2 = A0*A1*A3(temp=A3), B1 = A0*A2*A3(temp=A3*A2), B0 = A1*A2*A3(temp=A3*A2*A1)
