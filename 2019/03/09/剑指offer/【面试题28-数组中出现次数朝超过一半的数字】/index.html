【面试题28-数组中出现次数朝超过一半的数字】

Mar 9, 2019

剑指offer第二十八题。

题目描述

数组中有一个数字出现的次数超过数组长度的一半，请找出这个数字。例如输入一个长度为9的数组{1,2,3,2,2,2,5,4,2}。由于数字2在数组中出现了5次，超过数组长度的一半，因此输出2。如果不存在则输出0。

解题思路

因为全是数字，那么我开一个长度为10的数组，遍历原数组，根据计数的排序的思想，将新数组对应的索引值加一。遍历过程中发现某一个大于一半了就停止。
