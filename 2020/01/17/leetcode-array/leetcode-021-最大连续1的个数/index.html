解题思路

这一题的思路就是遍历数组，如果为1就不停计数，遇到0就中断掉，不断记录计数值从而找出最大值。

  • 时间复杂度：O(N)。N 值是数组的长度。
  • 空间复杂度：O(1)，仅仅使用了 count 和 max。
